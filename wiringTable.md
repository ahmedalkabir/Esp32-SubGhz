# Wiring Table
Following you see a Map of the Pins used in the Code.

## ESP32 - CC1101
| CC1101 | ESP32 |
| ------ | ------ |
| VCC | 3V3 |
| GND | GND |
| MISO | GPIO12 |
| MOSI | GPIO13 |
| SCK | GPIO14 |
| CSN | GPIO15 |
| GDO0 | GPIO2 |
| GDO2 | GPIO4 |

## ESP32 - MICRO-SD
| MICRO-SD | ESP32 |
| ------ | ------ |
| VCC | 3V3 |
| GND | GND |
| MISO | GPIO19 |
| MOSI | GPIO23 |
| SCK | GPIO18 |
| CSN | GPIO5 |

## ESP32 - TP4056
| TP4056 | ESP32 |
| ------ | ------ |
| OUT+ | 3V3 (Including a Switch) |
| OUT- | GND |

## TP4056 - 18650 BATTERY-HOLDER
| TP4056 | 18650 BATTERY-HOLDER |
| ------ | ------ |
| B+ | POSITIVE WIRE |
| B- | NEGATIVE WIRE |
